
# lcd, 7seg, ads_fsr
## ins
- push fsr
- display num to lcd
- display low mid high on ur choice

# getting start

run **index.py** file to run

## setting
- 7seg pin below
```py
  VCC:5v
  # 7seg 
  DIN: pin19
  CS : pin24
  CLK: pin23

  # adc, lcd
  SDA: pin3
  SCL: pin5

  # adc 
  using A0
```
## solution
  - press fsr to display num on 7seg and low mid high to lcd
